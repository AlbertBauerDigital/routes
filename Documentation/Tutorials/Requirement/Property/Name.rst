===================================
Name
===================================

We highly recommend to keep up the following naming convention if possible: **extension_controller-action**

.. code-block:: yaml
   :linenos:
   :emphasize-lines: 1

   demo_clients-index:
      path:         api/demo/clients
      controller:   LMS\Demo\Controller\ClientApiController::index

.. tip::
   **Required**: Yes
